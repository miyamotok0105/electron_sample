


## displayプロパティをflexにするとフレキシブルになる


```
.flex-container {
  display: flex;
  background-color: DodgerBlue;
}
```

flexコンテナのプロパティはこんだけある。


```
flex-direction
flex-wrap
flex-flow
justify-content
align-items
align-content
```
